# システム開発見積もり計算アプリ ユーザーガイド

## 概要

**目的**: システム開発プロジェクトの見積もりを人月単位で計算し、管理できます。

**アクセス**: ログイン不要で利用可能です。データはブラウザのローカルストレージに保存されます。

## Powered by Manus

このアプリケーションは最新のWeb技術で構築されています。

**技術スタック**:
- **フロントエンド**: React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
- **状態管理**: ブラウザローカルストレージによる永続化
- **デプロイ**: 自動スケーリングインフラストラクチャとグローバルCDN

## アプリケーションの使い方

### 見積もりプロジェクトの作成

ホーム画面で「新規作成」ボタンをクリックすると、見積もり編集画面が開きます。プロジェクト名を入力し、作業項目を追加していきます。「追加」ボタンをクリックすると新しい作業項目の行が追加されます。各作業項目には、作業項目名、職種、人月を入力します。職種はドロップダウンから選択でき、人月は数値で入力します。入力した内容は右側のサマリーパネルにリアルタイムで反映され、合計人月と合計金額が自動計算されます。

### テンプレートの活用

見積もり編集画面で「テンプレート」ボタンをクリックすると、事前に登録されたテンプレートから作業項目を一括で追加できます。標準的なWebアプリケーション開発のテンプレートが初期設定されており、要件定義から各種テストまでの工程が含まれています。テンプレートを選択すると、その作業項目が現在の見積もりに追加されます。

### 職種別単価の設定

右上の「設定」ボタンをクリックして設定画面を開きます。「職種別単価」タブでは、プロジェクトマネージャー、システムエンジニア、プログラマー、テスターなどの職種ごとに人月単価を設定できます。「職種を追加」ボタンで新しい職種を追加し、職種名と人月単価を入力します。設定した単価は見積もり計算に使用されます。

## アプリケーションの管理

### プロジェクト管理

ホーム画面では、作成したすべての見積もりプロジェクトが一覧表示されます。各プロジェクトカードには、プロジェクト名、作成日、作業項目数が表示されます。「編集」ボタンで既存のプロジェクトを編集でき、ゴミ箱アイコンでプロジェクトを削除できます。

### テンプレート管理

設定画面の「テンプレート」タブでは、よく使う作業項目のセットをテンプレートとして保存できます。「テンプレートを追加」ボタンをクリックすると、テンプレート編集ダイアログが開きます。テンプレート名を入力し、作業項目を追加していきます。保存したテンプレートは見積もり作成時に利用できます。

### データの保存

すべてのデータはブラウザのローカルストレージに保存されます。見積もり編集画面で「保存」ボタンをクリックすると、プロジェクトが保存されホーム画面に戻ります。設定画面でも「保存」ボタンをクリックすることで、職種別単価やテンプレートの変更が保存されます。
